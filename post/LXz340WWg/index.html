C++代码中的注释

在网络上有一句话：所有的程序员都讨厌两件事，1.别人不写注释 2.自己写注释。
但是为了自己以后能够读懂自己的代码和在比赛中能够及时发现自己的错误地方还是忍一忍写了吧，也可以用注释来调试自己的代码。
C++ 支持单行注释和多行注释。注释中的所有字符会被 C++ 编译器忽略。这也是我们用来调试代码的一个方法。

  • 不加注释：
  • 加了注释后：

    一般的编译器会让注释了的语句颜色不一样都可以区分出来的。
  • 一般常用的注释语法：
    语法1(单行注释)：
    以 // 开始，直到行末为止
#include<iostream>
using namespace std;

int main()
{
	cout<<"不被注释"<<endl;
	cout<<"被注释的一条"<<endl; //这是注释
	return 0;
}

语法2：
以 /* 开始，以 */ 终止。(多行注释)

/*作者：Cans518
   时间：2019.7.24
*/
#include<iostream>
using namespace std;

int main()
{
	cout<<"不被注释"<<endl;
	cout<<"被注释的一条"<<endl; 
	return 0;
}


这两种是常用的注释语法。但是还有一种，大家可以取网上取查一查。
